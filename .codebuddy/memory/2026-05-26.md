## 修复：IDE 拖拽文本到 Smart Copilot 失败（2026-05-26）
- **根因**：`BypassWindowManagerHint` 在 macOS 上绕过 WindowServer，导致拖拽事件无法被路由到 Qt 窗口
- **修复**：将 `AICardWindow` 和 `AgentWorkspace` 的 `BypassWindowManagerHint` 替换为 `Tool`（macOS NSPanel）+ `WA_ShowWithoutActivating`
- **效果**：保持浮窗置顶 + 不抢焦点的同时，恢复拖拽事件支持

## 修复：Smart Copilot 失焦后自动消失 + 拖拽问题（2026-05-26）
- **根因**：macOS 上 `Qt.Tool`（NSPanel）默认 `hidesOnDeactivate=True`，父窗口失焦时自动隐藏
- **最终方案**：不用 `Qt.Tool`，只用 `FramelessWindowHint | WindowStaysOnTopHint` + `WA_ShowWithoutActivating`
  - `Qt.Tool` + `hidesOnDeactivate=False` → 拖拽完全失效
  - `WindowDoesNotAcceptFocus` → 文本输入和拖拽都失效
  - 不用 `Tool` → 常驻+拖拽+输入全支持
- **IDE 编辑区无法拖拽**：Monaco Editor 拖拽是内部协议不经过系统 DnD，所有外部应用都无法接收
- **新增「📋 粘贴剪贴板」按钮**：AICardWindow 和 AgentWorkspace 都加了，用户 Cmd+C 后点击即可
