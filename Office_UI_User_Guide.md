## 一、主题切换

### 1.1 如何切换主题

1. 点击卡片右上角的 **⚙️ 设置** 按钮
2. 在设置对话框中找到 **"主题"** 选项
3. 选择你想要的主题：
   - **暗色主题** - 适合夜间或暗色环境
   - **亮色主题** - 适合会议室、居家办公
   - **办公主题** - Word蓝色风格，适合正式办公
4. 点击 **"应用"** 按钮

### 1.2 主题效果

| 主题 | 背景色 | 文字色 | 适用场景 |
|------|--------|--------|----------|
| 暗色 | 深灰 #2b2b2b | 白色 | 夜间办公、暗色环境 |
| 亮色 | 白色 #ffffff | 深灰 | 会议室、居家、白天 |
| 办公 | 浅灰 #f5f5f5 | 深蓝 | 正式办公、文档处理 |

### 1.3 快捷切换

使用快捷键 `Cmd+Shift+T` 可快速打开主题切换菜单。

## 二、快捷键

### 2.1 常用快捷键

| 快捷键 | 功能 | 使用场景 |
|--------|------|----------|
| `Cmd+Shift+Space` | 唤醒/隐藏卡片 | 随时调出AI助手 |
| `Cmd+Shift+T` | 翻译模式 | 快速翻译选中文本 |
| `Cmd+Shift+P` | 润色模式 | 润色优化文本 |
| `Cmd+Shift+R` | 文档修订模式 | 修订Word/PPT文档 |
| `Cmd+Shift+B` | 批量处理模式 | 批量处理多个文件 |
| `Cmd+Shift+S` | 打开设置 | 修改配置 |

### 2.2 如何自定义快捷键

1. 打开设置 (`Cmd+Shift+S`)
2. 切换到 **"快捷键"** 标签页
3. 点击要修改的快捷键
4. 按下新的快捷键组合
5. 点击 **"保存"**

### 2.3 快捷键冲突

如果快捷键被系统或其他应用占用：
- 会显示 ⚠️ 冲突提示
- 建议更换其他组合键

## 三、文件拖拽

> ✅ **已验证**：文件拖拽功能已修复并测试通过，支持 .md/.txt/.docx/.pptx 等格式

### 3.1 如何拖拽文件

1. 在 Finder 中选中文件
2. 按住鼠标左键拖动到卡片上
3. 松开鼠标完成拖入

**注意事项**：
- 拖拽过程中窗口不会消失（已修复 macOS 焦点丢失问题）
- 支持从 Finder 直接拖拽文件到卡片任意位置

### 3.2 支持的文件类型

| 文件类型 | 扩展名 | 拖入后效果 |
|----------|--------|------------|
| Word文档 | .docx | 自动进入文档修订模式 |
| PPT文档 | .pptx | 自动进入PPT处理模式 |
| PDF文档 | .pdf | 自动进入PDF处理模式 |
| 文本文件 | .txt | 自动进入文本处理模式 |
| Markdown | .md | 自动进入Markdown模式 |

### 3.3 拖拽后操作

拖入文件后会自动：
1. 显示文件信息（名称、大小、类型）
2. 进入对应的处理模式
3. 显示处理选项

### 3.4 批量拖拽

- 可以一次拖入多个文件
- 自动进入批量处理模式
- 显示处理队列和进度

## 四、右键菜单

### 4.1 文本右键菜单

1. 在任意软件中选中文本
2. 双击右键唤出卡片
3. 在卡片上点击右键
4. 选择操作：
   - **翻译** - 翻译选中文本
   - **润色** - 润色优化文本
   - **代码解析** - 解析代码含义
   - **复制结果** - 复制AI结果

### 4.2 文件右键菜单

1. 拖入文件后
2. 在文件卡片上点击右键
3. 选择操作：
   - **打开文件** - 用默认应用打开
   - **查看信息** - 查看文件详细信息
   - **移除文件** - 从列表中移除

### 4.3 代码右键菜单

1. 在IDE中选中代码
2. 拖入卡片后点击右键
3. 选择操作：
   - **代码解释** - 解释代码功能
   - **代码优化** - 优化代码结构
   - **添加注释** - 自动生成注释

## 五、设置对话框

### 5.1 如何打开设置

- 方式一：点击卡片右上角 **⚙️ 设置** 按钮
- 方式二：快捷键 `Cmd+Shift+S`

### 5.2 设置选项

#### 基础设置
- **主题** - 选择暗色/亮色/办公主题
- **字体大小** - 调整界面字体大小（8-24px）
- **语言** - 选择界面语言

#### AI设置
- **AI模型** - 选择MiniMax或Ollama
- **API Key** - 配置API密钥
- **温度** - 调整AI创造性（0-1）

#### 快捷键设置
- 查看所有快捷键
- 自定义快捷键
- 恢复默认设置

#### 高级设置
- **翻译记忆** - 启用/禁用翻译记忆
- **术语库** - 启用/禁用术语库
- **自动保存** - 启用/禁用自动保存

### 5.3 保存设置

修改后点击 **"保存"** 按钮，设置立即生效。

## 七、批量处理

### 7.1 如何批量处理

1. 拖入多个文件，或
2. 点击 **"批量处理"** 按钮
3. 在弹出的对话框中添加文件
4. 选择处理模式：
   - **批量翻译** - 翻译所有文件
   - **批量润色** - 润色所有文件
   - **批量修订** - 修订所有文档
5. 点击 **"开始处理"**

### 7.2 处理队列

- 显示待处理文件列表
- 显示每个文件状态（等待/处理中/完成/失败）
- 支持暂停/继续/取消

### 7.3 结果导出

处理完成后：
- 自动保存到原目录
- 文件名添加 `_processed` 后缀
- 支持批量导出

## 八、术语库管理

### 8.1 如何打开术语库

1. 打开设置 (`Cmd+Shift+S`)
2. 切换到 **"术语库"** 标签页

### 8.2 添加术语

1. 点击 **"添加术语"** 按钮
2. 填写：
   - **源术语** - 原文词汇
   - **目标术语** - 翻译/解释
   - **分类** - 术语分类（技术/商务/学术等）
   - **备注** - 补充说明
3. 点击 **"保存"**

### 8.3 搜索术语

- 在搜索框输入关键词
- 支持模糊搜索
- 显示匹配结果列表

### 8.4 导入导出

- **导入** - 支持JSON、TMX格式
- **导出** - 支持JSON、TMX、CSV格式

### 8.5 术语库应用

启用术语库后：
- 翻译时自动匹配术语
- 保持术语一致性
- 提高翻译质量

## 九、翻译记忆

### 9.1 什么是翻译记忆

- 自动保存翻译过的句子
- 相似句子自动匹配
- 避免重复翻译

### 9.2 如何使用

1. 确保翻译记忆已启用（设置中）
2. 正常使用翻译功能
3. 系统自动保存翻译记录

### 9.3 搜索记忆

1. 打开术语库对话框
2. 切换到 **"翻译记忆"** 标签
3. 输入要搜索的文本
4. 显示相似翻译结果

### 9.4 记忆管理

- **查看** - 浏览所有翻译记录
- **删除** - 删除错误记录
- **导出** - 导出为TMX格式
- **清空** - 清空所有记忆

### 9.5 应用场景

- **重复文本** - 自动使用之前的翻译
- **相似句子** - 提供参考翻译
- **术语一致** - 保持翻译一致性

## 十、常见问题

### 10.1 主题切换后没效果？

- 检查是否点击了"应用"按钮
- 尝试重启应用

### 10.2 快捷键不工作？

- 检查是否与其他应用冲突
- 检查系统偏好设置中的快捷键
- 尝试更换快捷键组合

### 10.3 文件拖拽无反应？

- 检查文件类型是否支持
- 检查辅助功能权限
- 尝试重新拖入

### 10.4 批量处理卡住？

- 检查网络连接
- 检查API Key是否有效
- 尝试取消后重新开始

### 10.5 术语库导入失败？

- 检查文件格式是否正确
- 检查文件编码是否为UTF-8
- 尝试使用JSON格式
