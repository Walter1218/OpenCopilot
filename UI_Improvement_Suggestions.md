## 一、当前 UI 架构分析

### 1.1 技术栈
- **GUI框架**: PyQt6
- **图形效果**: QGraphicsDropShadowEffect
- **光标特效**: 自定义 CursorOverlay 层
- **鼠标监听**: pynput 库

### 1.2 当前 UI 结构
```
smart_copilot.py
├── AICardWindow (主悬浮卡片)
│   ├── 标题栏 (状态指示灯 + 设置 + 关闭)
│   ├── Tab1: 快捷划词
│   │   ├── IDE/浏览器状态栏
│   │   ├── 快捷指令按钮栏
│   │   ├── 自定义指令输入栏
│   │   ├── 结果显示区
│   │   └── 操作按钮栏
│   └── Tab2: 连续对话
│       ├── 对话显示区
│       └── 输入发送栏
├── SettingsDialog (设置对话框)
└── CursorOverlay (光标特效层)
```

## 二、UI 改进建议

### 2.1 视觉设计优化

#### 2.1.1 主题系统
| 项目 | 当前状态 | 改进建议 |
|------|----------|----------|
| 颜色方案 | 固定暗色主题 | 支持明暗主题切换，自定义主题色 |
| 字体 | 系统默认 | 支持字体选择和大小调整 |
| 透明度 | 固定值 | 支持透明度滑块调节 |
| 圆角 | 12px 固定 | 支持圆角大小调整 |

#### 2.1.2 卡片视觉升级

#### 2.1.3 光标特效增强
| 特效 | 当前状态 | 改进建议 |

## 三、实施优先级

### 3.1 高优先级（P0）
1. **主题系统** - 提升用户体验的基础
2. **快捷键系统** - 提高操作效率
3. **文档处理专用界面** - 办公场景核心需求
4. **文件拖拽区** - 简化文件处理流程

### 3.2 中优先级（P1）
1. **光标特效增强** - 提升品牌辨识度
2. **上下文菜单** - 丰富交互方式
3. **翻译专用界面** - 专业翻译需求
4. **个性化设置** - 满足不同用户偏好

### 3.3 低优先级（P2）
1. **手势操作** - 高级用户需求
2. **批量处理界面** - 企业级需求
3. **毛玻璃效果** - 视觉优化
4. **动画效果** - 用户体验优化
