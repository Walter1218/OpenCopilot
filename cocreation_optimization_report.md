# PPT共创功能优化报告

## 优化目标

1. 优化转图表操作和图片操作，让它们能稳定生成高质量符合预期的产物
2. 实现Goal模式，利用AI生成prompt不断尝试产出正确的产物

## 优化前问题

### 问题1：转图表操作字段名不一致
- **现象**：AI返回的字段名与测试期望不一致
  - `columns` vs `headers`
  - `labels` vs `categories`
  - `steps` vs `nodes`
- **影响**：导致质量检查失败，通过率仅66.7%

### 问题2：图片操作缺少字段
- **现象**：AI未返回`image_desc`字段
- **影响**：导致图片操作测试失败

## 优化方案

### 方案1：优化Prompt
在prompt中明确指定字段名和数据结构：

```python
# 转表格
{
  "action": "update",
  "field": "table_data",
  "value": {
    "columns": ["列名1", "列名2"],
    "rows": [["值1", "值2"]]
  }
}

# 转图表
{
  "action": "update",
  "field": "chart_data",
  "value": {
    "labels": ["标签1", "标签2"],
    "datasets": [{"label": "系列1", "data": [100, 200]}]
  }
}

# 转流程图
{
  "action": "update",
  "field": "flowchart_data",
  "value": {
    "title": "流程图标题",
    "steps": ["步骤1", "步骤2"]
  }
}

# 添加配图描述
{
  "action": "update",
  "field": "image_desc",
  "value": "图片描述内容"
}
```

### 方案2：实现Goal模式
Goal模式的核心思想：
1. 定义目标（goal）
2. AI生成prompt
3. 执行并验证结果
4. 如果不符合目标，自动调整prompt重试
5. 直到达到目标或用户强制终止
