# SmartLock Pro 智能门锁产品规格书

## 1. 产品型号

| 型号 | 配色 | 售价 | 适用门厚 |
|------|------|------|----------|
| SL-P100 | 星空灰 | 1999 元 | 35-55mm |
| SL-P200 | 陨石黑 | 2499 元 | 40-60mm |
| SL-P300 | 流光银 | 2999 元 | 45-110mm |

全系产品提供 3 年整机质保，电池续航 12 个月。

## 2. 开锁方式

SL-P100 支持指纹（100 枚）+ 密码（50 组）+ IC 卡（20 张）。
SL-P200 新增 3D 人脸识别（5 组）+ 指静脉识别（50 组）+ 远程临时密码。
SL-P300 在前代基础上新增掌静脉识别（10 组）+ Apple Home Key 支持。

## 3. 安全特性

- C 级锁芯（全系标配）
- 半导体指纹传感器（拒真率 < 0.01%）
- 防特斯拉线圈攻击（SL-P200 / SL-P300）
- 防撬报警（全系）
- 门未关严提醒（全系）
- 胁迫指纹报警（SL-P200 / SL-P300）

## 4. 智能联动

支持通过 Wi-Fi（2.4GHz）接入米家 App 和 Apple HomeKit。SL-P300 额外支持 Thread 协议和 Matter 标准，可接入 Google Home 和 Amazon Alexa。
